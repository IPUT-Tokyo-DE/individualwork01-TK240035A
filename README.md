# Circle Ball and Triangle Bomb

## ゲームの内容
プレイヤー（四角）を操作して落ちてくるボールをキャッチするスコアアタックゲーム。三角を誤ってとってしまうとスコアが0に戻ってしまうので注意が必要。

![画面イメージ](docs/images/game_image01.png)

## 操作説明
例
- 矢印キー左右で四角を動かせます。操作はそれだけです。
